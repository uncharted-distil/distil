Step 1 - Select data

To begin, you need to find a dataset relevant to your problem. You can search for datasets from the Distil home page or the Select Data view.

Overview

In the Select Data view, you can search for datasets using keywords (e.g. Middle East) or by expressing your intent in natural language (e.g. Predict Middle East Conflict).

Search for a dataset

To find datasets related to Middle East attributes:
  1. Enter Middle East in the Search field on the Home page or the Select Data view. As you type, Distil returns each dataset that contains the keyword in its list of features or full description.
  2. Review the acled_clean result to determine its relevancy.
    • The Top Features, which indicate the most important features (columns), list Country_ISO_Number and several other quantitative features such as Event_Date.
      Note: Features are listed in order of importance, a measure of how much influence a feature has on other features in the dataset.
    • The Topics, inferred from text in the dataset, indicates that the dataset is about military conflict.
  3. Click More Details to see a complete summary.
  4. Click Name: acled_clean to choose the ACLED dataset for further inspection.
