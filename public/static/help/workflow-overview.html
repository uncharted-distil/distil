Workflow Overview

Use Distil to construct predictive models of automotive data.

In this workflow, you will learn how to find a relevant dataset and guide Distil to build a predictive model. Your task in this scenario is to categorize and predict conflict events from Middle East conflict report data. The workflow has five main steps:

  1. Select a dataset
  2. Select a target feature
  3. Create the models
  4. Join a dataset
  5. View the models
